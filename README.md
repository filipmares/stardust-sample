# Stardust Sample

Sample started project with React, Typescript and [Stardust](https://stardust-ui.github.io/react/).

## Getting Started

### Prerequisites
Install [yarn](https://yarnpkg.com/en/) for dependency management.
```
brew install yarn
```

### Installing

```
yarn install
```

### Running (scripts)
Start local server

```
yarn star
```

Start watch task
```
yarn watch
```

Build
```
yarn build
```
